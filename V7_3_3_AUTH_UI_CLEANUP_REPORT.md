# v7.3.3 인증/헤더 UI 정리 보고서

## 반영 내용
1. 상단 헤더에서 `내 계정` 버튼 제거
   - 로그인 후 일반 회원은 `회원` 버튼만 표시
   - 로그인 후 관리자도 `관리자` 버튼만 표시
   - 두 버튼 모두 `/account`로 이동

2. 홈 대시보드 `오답 복습 시작` 버튼 텍스트를 흰색으로 고정

3. 로그인/회원가입 화면 개선
   - 이미 로그인된 상태에서는 이메일/비밀번호 입력 폼과 로그인 버튼을 숨김
   - 대신 현재 로그인 계정, 이용 상태, 바로가기 버튼을 보여줌
   - 마이페이지, 문제 풀기, 관리자 대시보드(관리자만), 로그아웃 버튼 제공

4. 마이페이지의 `관리자 대시보드` 버튼 텍스트를 흰색으로 고정

## 기대 효과
- 헤더가 더 단순해집니다.
- 로그인 성공 후 불필요한 입력 폼이 사라져 상태가 더 명확해집니다.
- 관리자/일반 회원 모두 자신의 이동 경로를 쉽게 이해할 수 있습니다.

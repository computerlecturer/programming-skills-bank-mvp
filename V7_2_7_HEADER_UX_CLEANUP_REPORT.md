# v7.2.7 헤더 UX 정리 보고서

## 목적
v7.2.6에서 로그인 상태와 관리자 권한 표시를 강화하면서 상단 헤더가 과밀해지고, 메뉴 줄바꿈과 시각적 혼잡이 발생했습니다. v7.2.7에서는 로그인 상태는 유지하되 헤더의 가독성과 정보 구조를 정리했습니다.

## 반영 내용
1. 헤더 메뉴 줄바꿈 방지
   - 메뉴 링크에 `whitespace-nowrap` 적용
   - 데스크톱 메뉴는 `lg:flex` 이상에서만 표시

2. 가운데 메뉴 축소
   - `분석` 제거
   - `내 계정` 제거
   - 유지 메뉴: 과목, 모의고사, 오답, 보관함, 베타 안내

3. 로그인 상태 표시 축소
   - 이메일 직접 노출 제거
   - 일반 회원은 `회원` 배지
   - 관리자 계정은 `관리자` 배지만 표시
   - 배지를 클릭하면 `/account`로 이동

4. 오른쪽 버튼 정리
   - 로그인 상태: `[회원/관리자] [문제 풀기]`
   - 미로그인 상태: `[로그인] [무료 시작]`
   - 로그인 상태에서 별도 `내 계정` 버튼 제거

5. 로고 영역 반응형 정리
   - 서비스명은 말줄임 처리
   - 부제목은 넓은 화면(`xl`)에서만 표시

## 기대 효과
- 상단 메뉴가 한 줄로 유지되어 가독성이 개선됩니다.
- 관리자/회원 상태는 확인 가능하되 헤더의 주인공이 되지 않습니다.
- 핵심 CTA인 `문제 풀기`가 더 명확하게 보입니다.
- 헤더가 다시 가볍고 전문적인 인상으로 정리됩니다.

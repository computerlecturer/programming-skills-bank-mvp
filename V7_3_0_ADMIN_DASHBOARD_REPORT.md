# v7.3.0 관리자 대시보드 추가 보고서

## 목적
v7.2.9까지 오류 제보 저장, 관리자 목록 확인, 문제 원문/정답/해설 검토 기능이 완료되었습니다. v7.3.0에서는 관리자 기능의 입구가 되는 `/admin` 대시보드를 추가했습니다.

## 추가된 화면
- `/admin`
  - 관리자 전용 대시보드
  - 오류 제보, 회원, 문제은행, AI 테스트, 베타 운영 상태를 한 화면에서 확인

## 대시보드 표시 항목
1. 오류 제보 운영 현황
   - 전체 오류 제보 수
   - 오늘 접수 수
   - 미처리 수
   - 수정 필요 수
   - 상태별 수
   - 최근 오류 제보
   - 수정 필요 제보

2. 회원 현황
   - 전체 회원 수
   - 오늘 가입 수
   - 일반 회원 수
   - 관리자 수
   - 최근 가입 회원

3. 문제은행 현황
   - 전체 문항 수
   - SQL/Python/Java/Linux 과목별 문항 수

4. AI 테스트 현황
   - 오늘 AI 테스트 수
   - 월간 AI 테스트 수
   - 전체 AI 사용 로그 수

5. 베타 운영·시스템 상태
   - 로그인 방식
   - 결제 비활성화 상태
   - AI 관리자 테스트 상태
   - 오답노트/보관함 localStorage 저장 안내
   - 현재 버전

## Supabase SQL
다음 파일을 SQL Editor에서 1회 실행해야 회원 현황과 AI 테스트 현황까지 정상 표시됩니다.

`supabase/migrations/v7_3_0_admin_dashboard_policies.sql`

해당 SQL은 관리자 계정이 profiles와 ai_usage_logs를 조회할 수 있도록 RLS 정책을 보강합니다.

## 기존 기능 유지
- `/admin/reports` 오류 제보 관리 화면 유지
- 기존 오른쪽 오류 제보 패널 유지
- 일반 회원/관리자 권한 구분 유지
- 오답노트/보관함 localStorage 저장 정책 유지

# Linux 학습하기 문제풀이형 개편 보고서

## 수정 목적

Linux 문제에서 학생이 `학습하기`를 눌렀을 때 단순 이론 설명만 보는 것이 아니라, 다시 문제로 돌아가 스스로 풀 수 있도록 학습하기 구조를 개편했습니다.

## 주요 수정 사항

1. Linux 120문항의 학습하기 데이터를 전면 재구성했습니다.
   - 명령어 기능형
   - 디렉터리 이동형
   - 절대경로·상대경로 계산형
   - 파일·디렉터리 조회형
   - 파일 관리형
   - 파일 내용 확인형
   - chmod 권한 계산형
   - 관리자 권한·도움말·환경 변수형
   - 프로세스·디스크·네트워크형
   - 실전 혼합형

2. Linux 학습하기 팝업에 `문제에서 먼저 봐야 할 단서` 영역을 추가했습니다.
   - 현재 문제의 정답을 직접 노출하기보다, 어떤 단서를 봐야 하는지 안내합니다.
   - 경로 문제는 공통 경로, `..` 개수, 절대/상대 여부를 확인하도록 구성했습니다.
   - 명령어 문제는 핵심 동사와 대상 파일·디렉터리를 확인하도록 구성했습니다.

3. Linux 학습하기 팝업 하단에 `Linux 필수 명령어 도감`을 추가했습니다.
   - 위치·기본
   - 파일·디렉터리 관리
   - 파일 내용 확인
   - 권한·도움말·환경
   - 시스템·네트워크·검색
   - 경로 기호 빠른 정리
   - 권한 숫자 빠른 정리

4. Linux 학습하기 사용법 문구를 별도 적용했습니다.
   - 정답을 바로 알려주는 것이 아니라 단서, 풀이 순서, 실수 포인트, 명령어 도감을 통해 스스로 풀 수 있게 돕는 방식으로 안내합니다.

5. 기존 검수에서 확인된 SQL 객관식 선택지 전달 누락 가능성을 함께 보완했습니다.
   - `choiceOptions`를 정규화된 Question 객체에 전달하도록 수정했습니다.

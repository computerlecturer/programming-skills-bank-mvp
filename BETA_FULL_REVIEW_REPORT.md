## 이번 베타 파일에 반영한 추가 수정

1. SQL 23번은 기존 빈칸 위치상 `NOT 지역 = '부산'` 형태가 학생에게 어색하게 보일 수 있어, `지역 (   ) '부산'` 형태의 비교 연산자 문제로 수정했습니다. 정답은 `<>`이며, `!=`도 인정하도록 했습니다.
2. Java 6번과 7번은 단순 키워드 암기형 문제를 제거하고, 각각 static 변수 추적 문제와 print/println 실행 결과 문제로 교체한 최신 데이터를 반영했습니다.
3. 전체 620문항에 대해 학습하기 이론 데이터가 연결되어 있는지 확인했습니다.
4. result-table 문제는 정답 컬럼/행 구조가 존재하는지 점검했습니다.

## 베타 테스트 시 우선 확인할 부분

- SQL 결과표 작성형에서 학생이 직접 열/행을 구성하는 흐름이 불편하지 않은지 확인해야 합니다.
- Java/Python 출력 결과 문제는 줄바꿈 채점이 엄격하므로, 학생 안내 문구가 충분한지 확인해야 합니다.
- Linux 명령어 문제는 공백이 문법상 중요할 수 있으므로, `cd ..`와 `cd..`처럼 의미가 달라지는 답안은 구분됩니다.
- 학습하기 팝업은 정답 힌트가 아니라 개념 설명 중심이므로, 학생이 문제를 풀기 전 참고자료로 이해하는지 확인하면 좋습니다.

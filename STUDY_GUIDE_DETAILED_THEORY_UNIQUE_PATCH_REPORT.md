# 학습하기 상세 이론 문항별 고유화 패치 보고서

## 반영 내용

- SQL 200문항, Python 150문항, Java 150문항의 `이 문제를 풀기 위한 자세한 이론 보기` 영역을 문항별 상세 이론으로 재작성했습니다.
- 각 문항의 실제 지문, 코드, 표, 단원, 유형을 읽어 `문제에서 먼저 봐야 할 단서`, `풀이 순서`, `자주 하는 실수`, `자세한 이론`을 다르게 구성했습니다.
- SQL/Java/Python도 학습 팝업에서 `문제에서 먼저 봐야 할 단서` 섹션이 표시되도록 UI를 수정했습니다.
- 비슷한 예시는 현재 문제의 정답을 직접 노출하지 않도록 다른 값과 다른 예시로 구성했습니다.
- 상세보기 버튼 문구를 `이 문제를 풀기 위한 자세한 이론 보기`로 변경했습니다.

## 검증 요약

| 과목 | 문항 수 | 상세 이론 고유 개수 | 단서 섹션 고유 개수 |
|---|---:|---:|---:|
| SQL | 200 | 200 | 200 |
| Python | 150 | 150 | 150 |
| Java | 150 | 150 | 150 |
| Linux | 120 | 10 | 10 |

※ Linux는 직전 패치에서 유형별 문제풀이형 학습하기와 명령어 도감 구조를 반영했기 때문에 이번 패치에서는 SQL/Python/Java 중심으로 보강했습니다.

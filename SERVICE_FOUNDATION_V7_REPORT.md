# v7 서비스 기초 버전 수정 보고서

## 기준 파일

- v6 학습하기 미니 강의 검수본을 기준으로 서비스화 구조를 추가했습니다.
- 문제 본문, 정답, 해설, 학습하기 콘텐츠는 수정하지 않았습니다.

## 주요 수정 내용

1. 무료/유료 정책 반영
   - 과목별 50문항 무료
   - 51번 이후 문항 잠금
   - 전체 문제은행 30일 이용권 정책 상수화

2. 회원 기능 기초 구조
   - `/login` 로그인 화면 추가
   - `/signup` 회원가입 화면 추가
   - `/account` 마이페이지 추가
   - 이메일/구글/카카오 로그인 UI 준비
   - v7에서는 Supabase 연결 전 로컬 테스트 세션으로 동작

3. 이용권 권한 구조
   - 30일 이용권 상태 확인 함수 추가
   - 테스트용 이용권 부여/해제 버튼 추가
   - 유료 이용권 활성 시 51번 이후 문제 접근 가능

4. 유료 기능 게이트
   - 모의고사
   - 오답노트
   - 약점 분석
   - 학습 보관함

5. AI 문제 변형 UI 준비
   - 정답을 맞힌 문제에서만 활성화
   - 30일 이용권 활성 사용자에게만 사용 가능
   - 하루 20회 남은 횟수 표시
   - 현재 난이도 유지 / 난이도 높이기 두 모드 버튼 배치
   - 실제 OpenAI API 생성은 v9 단계에서 연결 예정

6. DB 준비
   - `supabase/schema.sql` 추가
   - profiles, user_entitlements, payments, user_question_status, wrong_notes, user_memos, error_reports, ai_generated_questions, ai_usage_logs 테이블 초안 작성

## 아직 남은 작업

- 실제 Supabase Auth 연결
- 실제 카카오/구글 OAuth 키 연결
- 실제 결제 연동
- 결제 성공 웹훅 검증
- 학습기록 DB 저장 전환
- OpenAI API 기반 AI 문제 생성
- 관리자 페이지

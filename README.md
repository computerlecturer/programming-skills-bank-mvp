# 프로그래밍기능사 실기연구소 v7.4.8 서비스 UX 정리

2026 프로그래밍기능사 실기 대비 문제은행의 베타 회원제 서비스 버전입니다. 이번 버전은 v7.4.7의 학습모드/실전모드 기능을 유지하면서, 학습자 화면에서 상태가 겹쳐 보이지 않도록 문제풀이 UX를 정리한 개선본입니다.

## 포함 콘텐츠

- SQL 200문항
- Python 150문항
- Java 150문항
- Linux 120문항
- 전체 620문항
- 문항별 개념 보기 미니 강의 데이터 620개

## 베타 운영 정책

- 비회원은 과목별 50문항까지 무료로 이용할 수 있습니다.
- 회원가입한 사용자는 베타 기간 동안 전체 620문항을 무료로 이용할 수 있습니다.
- 결제 기능은 현재 비활성화되어 있습니다.
- AI 문제 변형 기능은 일반 회원에게 “서비스 준비중입니다”로 안내합니다.
- `profiles.role = 'admin'`인 관리자 계정만 정답을 맞힌 문제에서 AI 문제 변형 테스트를 할 수 있습니다.

## v7.4.8 주요 개선 사항

- 학습모드에서 `채점하기`와 `정답 보기`를 명확히 분리했습니다.
- 정답 보기를 누른 문제는 직접 정답 처리하지 않고 `답 봄`으로 별도 기록합니다.
- 정답 공개 카드와 오답 카드가 동시에 표시되지 않도록 결과 패널 표시 조건을 정리했습니다.
- 학습모드 오답 제출 시 내가 쓴 답, 정답, 해설을 한 카드 안에서 확인할 수 있게 정리했습니다.
- 실전모드에서는 힌트/정답 보기/개념 보기를 숨기고 제출 후 내가 쓴 답만 확인하도록 흐름을 단순화했습니다.
- AI 변형 문제 안내는 정답을 맞힌 뒤 접힘 패널로만 노출되도록 낮은 우선순위로 조정했습니다.
- 51번 이후 회원 전용 잠금 화면을 `무료 회원가입` 중심의 자연스러운 안내로 수정했습니다.
- 버튼 문구를 `정답 확인 → 채점하기`, `답 보기 → 정답 보기`, `학습하기 → 개념 보기`, `다시 입력 → 다시 풀기`, `다시 볼 문제 → 북마크` 기준으로 정리했습니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 다음 주소로 접속합니다.

```text
http://localhost:3000
```

## 환경변수

프로젝트 루트에 `.env.local`을 만들고 다음 값을 입력합니다.

```env
NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=sb_publishable_...
```

베타 테스트만 진행하는 경우 결제 키와 OpenAI 키는 비워두어도 됩니다.

## GitHub 적용 안내

기존 GitHub Desktop 프로젝트 폴더는 유지하고, 이 ZIP의 압축을 푼 뒤 내부 파일을 기존 프로젝트 폴더에 덮어쓰는 방식을 권장합니다. 기존 `.git`, `.env.local`, `node_modules`, `.next`는 건드리지 않습니다.

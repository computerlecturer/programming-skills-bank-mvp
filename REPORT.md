# v7.4.8 서비스 UX 정리 보고서

## 버전

- package.json: `0.7.4.8-service-ux-refine`
- SERVICE_VERSION: `v7.4.8-service-ux-refine`

## 수정 목적

v7.4.7에서 학습모드 답 보기와 실전모드 순차 풀이 기능은 추가되었으나, 실제 모바일 화면에서 정답 공개 카드, 오답 카드, AI 변형 안내가 동시에 노출되어 학습자가 현재 상태를 혼동할 수 있었습니다. v7.4.8은 학습자에게는 단순한 풀이 흐름을 제공하고, 서비스 제공자에게는 답 봄·오답·정답 데이터가 분리되어 쌓이도록 화면 상태를 정리한 버전입니다.

## 주요 반영 사항

1. 학습모드 상태 분리
   - `채점하기`와 `정답 보기` 문구 및 동작을 분리했습니다.
   - 정답 보기를 누르면 `답 봄`으로 기록되며 정답률에는 포함되지 않습니다.
   - 정답 공개 상태와 오답 제출 상태가 동시에 표시되지 않도록 `answerRevealSuppressed` 상태를 추가했습니다.

2. 결과 패널 정리
   - 정답, 오답, 정답 공개 중 하나의 결과 패널만 표시되도록 조건을 정리했습니다.
   - 학습모드 오답에서는 내가 쓴 답, 정답, 해설을 함께 보여줍니다.
   - 실전모드 오답에서는 정답과 해설을 숨기고 내가 쓴 답만 보여줍니다.

3. 버튼 문구 정리
   - `정답 확인` → `채점하기`
   - `답 보기` → `정답 보기`
   - `학습하기` → `개념 보기`
   - `다시 입력` → `다시 풀기`
   - `다시 볼 문제` → `북마크`

4. AI 변형 안내 위치 조정
   - 문제 풀이 중간에 항상 노출되던 AI 변형 패널을 제거했습니다.
   - 학습모드에서 정답을 맞힌 뒤에만 접힘 패널로 낮게 노출되도록 변경했습니다.

5. 회원 전용 잠금 화면 개선
   - 51번 이후 잠금 화면을 `무료 회원가입`과 `베타 전체 이용` 중심으로 수정했습니다.
   - 비회원 50문항 제한 정책은 그대로 유지했습니다.

## 검증 내용

- JSON 데이터 파싱 가능 여부 확인
- Python 59번·77번 기존 수정 유지 확인
- `.env.local`, `node_modules`, `.next`, `.git`, `tsconfig.tsbuildinfo`는 배포 ZIP에서 제외

## 추가 안내

현재 작업 환경에는 `node_modules`가 없어 `npm run build`는 실행하지 못했습니다. 기존 프로젝트 폴더에 덮어쓴 뒤 로컬에서 `npm install` 또는 기존 `node_modules`를 유지한 상태로 `npm run dev`를 실행해 화면을 확인하면 됩니다.

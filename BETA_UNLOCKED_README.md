# 베타 전체 공개 안내 파일 폐기

이 프로젝트는 v7부터 베타 전체 공개 모드를 종료하고 서비스 기초 구조로 전환되었습니다.

현재 기준:

```ts
export const BETA_UNLOCK_ALL = false;
export const FREE_LIMIT_PER_SUBJECT = 50;
```

- 과목별 50문항까지 무료 공개
- 51번 이후 문항은 30일 이용권 권한 필요
- 테스트용 30일 이용권은 `/account` 화면에서 로컬 세션으로 부여 가능

이 파일은 과거 베타 공개 버전과 구분하기 위한 기록용입니다.

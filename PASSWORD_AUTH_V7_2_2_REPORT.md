## 사용자 UX 개선

- 회원가입과 로그인을 명확히 분리했습니다.
- 정상/오류/중복 확인 메시지를 화면 하단에 명확히 표시합니다.
- 비밀번호 보기/숨기기 버튼을 추가했습니다.
- 중복 확인을 하지 않은 상태에서는 회원가입이 진행되지 않도록 했습니다.
- 소셜 로그인과 인증번호 흐름을 제거해 사용자가 어떤 방식으로 가입해야 하는지 헷갈리지 않게 정리했습니다.

## 주의사항

Supabase의 이메일 확인 설정이 켜져 있으면 회원가입 후 즉시 로그인되지 않을 수 있습니다. 베타 테스트에서는 `Confirm email`을 OFF로 설정하는 것을 권장합니다.

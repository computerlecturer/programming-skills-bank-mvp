# SQL 선택형 문항 정리 보고서

## 수정 목적

SQL 문제 중 <고르시오> 형태로 보기 4개를 제시하던 문항을 학생이 직접 SQL문, 키워드, 개념 설명을 작성하는 방식으로 정리했습니다.

## 수정 이유

프로그래밍기능사 실기 대비 플랫폼의 핵심은 짧은 코드 해석, SQL 작성, 빈칸 작성, 실행 결과 작성형 훈련입니다. 보기형 문제는 초보 학습에는 도움이 될 수 있으나, 실기 대비 화면에서는 정답 구조가 과도하게 노출되어 학습 효과와 실전성이 떨어질 수 있습니다.

## 수정 대상

다음 SQL 문항의 보기형 지문을 직접 작성형 또는 개념 서술형으로 변경했습니다.

6, 9, 15, 19, 28, 38, 48, 56, 66, 70, 76, 80, 83, 84, 86, 90, 99, 114, 125, 132, 141, 146, 150, 156, 166, 169, 177, 185, 189, 200

## 추가 반영

- 문제 유형 필터의 내부 코드 표시를 학생용 한글 문구로 변경했습니다.
- 학습 모드에서 [학습하기] 버튼을 제공하여 정답 힌트가 아닌 이론 설명을 볼 수 있게 했습니다.
- SQL 최종 검수 데이터 파일을 실제 앱 로딩 대상으로 지정했습니다.

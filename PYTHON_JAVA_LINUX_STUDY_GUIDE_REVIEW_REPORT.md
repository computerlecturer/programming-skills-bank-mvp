# Python·Java·Linux 학습하기/해설 보강 보고서

## 반영 범위

- Python 150문항 학습하기 전체 검토 및 보강
- Java 150문항 학습하기 전체 검토 및 보강
- Linux 120문항 학습하기 전체 검토 및 보강
- Python 12번 정답 해설 보강

## 주요 반영 내용

1. Python 12번 해설에 `type(a)`와 `.__name__`의 의미를 명확히 추가했다.
2. 학습하기 내용을 단순 개념 요약이 아니라 문제 풀이에 필요한 사전 지식 중심으로 확장했다.
3. 정답을 직접 노출하기보다, 학생이 문제를 풀 수 있도록 개념·풀이 순서·주의사항을 보강했다.
4. Python은 출력, 자료형, 리스트/슬라이싱, 튜플, 딕셔너리, 반복문, 조건문, 함수, 클래스 유형별로 이론을 보강했다.
5. Java는 출력, 자료형, String 비교, interface/implements, 상속/생성자, 조건문/반복문 흐름을 중심으로 보강했다.
6. Linux는 pwd, cd/상대경로, ls, 파일·디렉터리 관리, chmod 권한, grep/find, ps/top/kill, man/help 유형을 중심으로 보강했다.

## 검토 기준

- 문제를 풀기 전 학습하기만 읽어도 핵심 개념을 이해할 수 있는가
- 해설이 정답만 말하지 않고 왜 그렇게 되는지를 설명하는가
- 한국어 서술이 너무 짧거나 추상적이지 않은가
- 시험형 문제 풀이에 필요한 주의사항이 포함되어 있는가
